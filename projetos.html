Como adicionar projetos:

Para adicionar um novo projeto ao banco de dados, siga as instruções abaixo:

  1. Solicite permissão ao professor responsável.
  2. Inclua os dados na planilha na seguinte ordem:
    1. Link do repositório
    2. Descrição do projeto
    3. Nome do projeto
    4. Link da imagem no repositório
  3. Crie um arquivo README.md no repositório para melhor visualização.
